Backhub : Application d’archivage de données IT Service Management

Sept 2025 — En cours

Ce projet web s’inscrit dans un contexte de rationalisation de différentes plateformes ITSM. Backhub est une application permettant d’archiver les données des tickets d’incidents issus de ces plateformes dans une base de données MongoDB.

Un utilisateur peut ensuite consulter les données archivées grâce à une interface web. Celle-ci comprend une liste des tickets archivés, un historique des tickets consultés (implémenté avec Redis) ainsi que des fonctionnalités de recherche.

Lors du début du projet, j'ai fait le choix de générer l'interface web avec le moteur de template EJS pour terminer rapidement une première version de l'application, ne connaissant pas les solutions plus modernes mais complexes comme React ou Angular. Le prochain stade du développement est donc de remplacer EJS par React.

Amare : Application d'audit multi-référentiels des actifs d'entreprise

Jan 2025 — Avril 2025

Cette application web permet de comparer la cohérence des données des actifs d'entreprise répertoriés entre les plateformes ServiceNow et Tanium.

L’application consomme les API des plateformes, consolide les données, les analyse et les sauvegarde ensuite en base de données. Les résultats sont présentés à l’utilisateur via l’interface web sous forme d'un rapport. Ce rapport est une liste des actifs avec un badge d'approbation lorsque les données de l'actif sont cohérentes. La cohérence est calculée grâce à un algorithme utilisant la distance de Levenshtein.
